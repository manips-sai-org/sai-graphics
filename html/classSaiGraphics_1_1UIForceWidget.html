Detailed Description

A class to enable the application a force or moment to a robot or object object in the world by detecting which point on the robot/object is under the cursor when turned on (for example when clicking) and computing a force or moment proportional to the drag distance, with velocity based damping. The force/moment is rendered as a green/red line in the world.
